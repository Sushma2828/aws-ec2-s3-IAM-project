# AWS EC2 + S3 Static Website

## 📌 Project Description
This project demonstrates hosting a static website using Amazon EC2 and storing images in Amazon S3 with secure IAM role access.

## 🚀 Services Used
- Amazon EC2
- Amazon S3
- AWS IAM

## ⚙️ Steps Performed
1. Launched EC2 instance
2. Installed web server
3. Uploaded website files
4. Created S3 bucket and uploaded images
5. Configured IAM role
6. Connected S3 with EC2
7. Hosted website successfully
